# TaskList
    TaskList é uma aplicação simples de lista de tarefas desenvolvida em React.

#   Tecnologias Utilizadas

<ul>
    <li>React</li>
    <li>Typescript</li>
    <li>Sass</li>
</ul>    
    
# Funcionalidades
<ul>
    <li>Adicionar novas tarefas à lista.</li>
    <li>Remover tarefas existentes.</li>
</ul>
